1、单字符IO
	int getchar(void);
	功能：从键盘获取一个字符并返回
	参数：无
	返回值：成功 返回获取的键盘字母
			失败 EOF
		注意：EOF值为-1；
	注意：该代码会阻塞等待用户输入，任何输入都是按照
	ascii值输入，都是回车换行提交数据。
	int putchar(int c);
	功能：将参数c对应的字母输出到默认的输出设备显示器终端
	参数：c 要输出的字符
	返回值：成功 返回值就是输出的字符
			失败 EOF
	注意：putchar()函数不会阻塞，直接输出
练习：输入自己的姓名并打印输出
	  将自己的名字转换成大写字母输出
	  将自己的名字加密
2、字符串IO===》gets()/puts()
	char *gets(char *s); ==>void gets(void buff[]);
	功能：默认阻塞等待并从键盘获取用户任意输入的字符串
	参数：s buff 用于存储数据的内存区==》栈区
			char buff[128]={0};//定义一个字符串数组
	返回值：成功 返回接受到的字符串的地址
			错误 NULL
	int puts(const char *s);==>int puts(buff[]);
	功能：将制定的buff/s所在的内存区中的字符串打印输出
		  到屏幕终端
	参数：buff/s 要输出的数据
	返回值：成功 返回>=0的值 ==字符串的长度
			失败 EOF
	注意：puts输出时自带'\n'
练习：用gets()/puts()函数来获取用户信息并打印输出，提示用户输入#退出，否则继续
3、格式化IO ===》printf()/scanf()
	int scanf(const char *format, ...);
	功能：从键盘缓存区扫描数据并以制定格式收取概数据
		  该函数回族色等待用户输入，回车提交数据
	参数：format 输入数据的形式
		  ...	 可变长参数
		  format的格式如下："%[修饰符]格式符"
		  "%c" 按照单字符形式获取数据 == getchar()
		  "%d" 按照十进制整数形式获取数据
		  "%x"  "%f" "%s" "%o"
		  注意：以上格式可以有一个，也可以有多个，如果				  有多个，用空格或者制定格式隔开
	***	  输入的格式符最后一定不要有'\n'或''
		  原因：scanf会自动忽略空白符'\n''tab'
	***	  如果scanf都是以"%c"获取数据则回车也会接收
	***	  如果scanf是字符串，则可变参数部分的变量一般
		  数组，则不用取地址符号
	***	  scanf函数在"%s"只能获取空白符之前的第一个字
		  符串
	修饰符：n ==》表示数字宽度
	...可变长参数 必须和格式符一一对应
		同时必须保证输入的参数必须是地址
	返回值：成功 返回收到的数据单元的个数
			失败 EOF
			文件结尾 0
练习：设计一个程序可以获取用户输入的字符和数字，并完成一个简单的计算器
	int printf(const char *format, ...);
	功能：按照format指定的格式向屏幕打印输出
	参数：format 输出的格式，有如下格式
		"%[修饰符]格式符"
		"%d" 十进制整数输出
		"%c" ascii字符输出
		"%o" 八进制输出 ==》"%#o"自带格式
		"%x" 16进制输出 ==》"%#x"自带格式 0xac
		"%X" 16进制输出					  0XAC
		"%f" 单精度浮点数
		"%lf"双精度浮点数
		"%u" 无符号十进制整数
		"%s" 字符串输出
		修饰符 # 0 - + m .n h l
		# 表示自带格式输出 常用与8进制16进制
		0 输出域不够时补0
		- 输出过程中对于负数自带-
		+ 输出过程中对于整数自带+
		m 输出过程中数据的宽度
		.n输出过程中小数点后的有效数
			在字符串输出过程中指定字符的个数
		h 短整形
		l 长整形
