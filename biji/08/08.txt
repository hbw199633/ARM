C语言的控制语句
	分类：顺序语句、分支语句、循环语句
	顺序语句：
	从main主函数入口开始一行一行往下执行
分支语句：
	单分支：if...else
	多分支：switch...case
循环语句：
	for循环
	while循环
	do...while循环
	goto循环
单分支：if...else
格式：if(逻辑表达式)
	  {
		语句1：//逻辑表达式为真执行
	  }
	  else
	  {
		语句2：//逻辑假执行
	  }
		执行流程：
		首先判断逻辑能表达式是否为真，
		如果为真，则执行语句1；
		如果为假，则执行语句2；
	if(逻辑表达式1）
	{
		if(逻辑表达式2)
		{
		}
	}
	else
	{
		if(逻辑表达式3)
		{
		}
	}
说明：
	1、最简形式==》(a>b)?xxx:yyy
		条件表达式的一般行为：
		表达式1？表达式2：表达式3
	2、else 可以省略，if(逻辑表达式) {}
	3、{}可以省略，只有一条语句时方可
	4、支持阶梯嵌套
	5、嵌套不建议超过4层

多分支语句：
	switch...case
	格式：
	switch (整数表达式)
	{
		case 常量表达式1：语句1；break;
		case 常量表达式2：语句2；break;
		case 常量表达式3：语句3；break;
		case 常量表达式4：语句4；break;
		...
		default:
			语句;
	}
执行流程：
	计算整数表达式，在case部分的常量表达式中依次匹配，
	如果常量表达式的值与整数表达式完全匹配，则执行该
	case之后的语句,如果没有常量的表达式，则执行default
	语句，如果没有default语句，则分支判断结束
注意事项：
	1、case之后的常量表达式必须是值，并能匹配整数表达式，不能是一个字符串或者一个范围
	2、在控制过程中的break能唯一结束一次判断，并跳出
	switch语句，如果没有break,则语句执行到第一个swtich
	3、default可以没有，但如果匹配项目没有，则switch执行完毕后顺序执行
	4、switch支持嵌套

循环语句：
for while do...while goto
1、for循环
	标准格式：
	for(表达式1；表达式2；表达式3)
	{
		//循环体语句
	}
执行过程：
	1、首先执行表达式1，且只执行一次
	2、根据表达式2的逻辑值判断是否执行循环体，如果表
	   达式2逻辑值为真，则执行循环体，如果表达式2逻辑
	   值为假，则停止循环
	3、循环体语句执行完毕，默认执行表达式3
注意事项：
	1、循环体语句的{}可以省略，但是要求单条语句
	2、表达式之间用;隔开，不能用, . :
	3、表达式1可以省略,但;不能省略,必须提前执行
	   表达式3可以省略,但是必须在循环体中执行
	   表达式2可以省略,但;不能省略,表示死循环for(;;)
	4、for循环语句支持嵌套形式，建议不要超过4层循环
	5、for循环中的中断辅助控制break;关键字,只能跳出
	   一层循环	
	   for循环中的继续辅助控制continue;关键字,继续从
	   循环位置执行
	6、for循环的结尾部分没有；
练习：
1、用for写延时
2、用for写1到100之内能被3正除的数字
3、9*9乘法表
	while循环
	格式：while(表达式)
		  {
			循环体；
		  }
执行流程：
	首先判断逻辑表达式，如果为真则执行循环体，如果为假循环结束。
	表达式：可以是逻辑表达式,赋值表达式,关系表达式
注意事项：
	1、while循环()之后不能有；
	2、while的{}可以省略，要求循环体的语句是一条语句
	3、while循环的死循环while(1),但是尽量不要空转
	4、while支持嵌套
	5、while循环中的中断辅助控制break;关键字,只能跳出
	   一层循环	
	   while循环中的继续辅助控制continue;关键字,继续从
	   循环位置执行

	do...while循环语句
格式：
	do
	{
		循环体；
	}while(表达式)；
执行过程：
	先执行循环体语句，在判断逻辑表达式的值，如果为真，
	执行循环，如果为假，循环结束。循环至少执行一次
注意事项：
	1、while之后必须有；
	2、区别while和do...while的差异
		while    先判断后执行
		do..while先执行后判断
	3、do...while支持嵌套，建议不要超过4层
	4、在do...while中，break跳出，continue继续，
	   return结束函数
练习：
	用do...while完成1000以内能被4整除的数之和
goto循环   需要标号支持
	eg:定义标号
		flag: ===>标号 xxx:
			语句块;
		goto flag; ==》执行到该语句返回到上层flag标记
					   处继续执行‘
执行过程：
	按照顺序执行遇到的goto跳转执行
注意事项：
	1、不支持break,continue循环控制关键字，return可以
	   结束当前循环
	2、标号之后不能是变量定义
	3、goto语句不能使用在变量的定义和声明位置
	4、goto语句一般在编程过程中用于错误处理
